# How to update the website
There's a video walkthrough here: https://drive.google.com/file/d/14dihWx2dDgC6rEDPDquBoq3J3dDf-Czg/view

Rough breakdown:
1. The pages are split into different HTML pages above
2. There's a central stylesheet called styles.css
3. Images are uploaded into the same repo and referenced in the HTML files
4. Make changes to the repo (you can edit the files with the ✏️ in the top right on individual files) and commit your changes (green button in the top right)
5. Wait around 20 seconds once these changes are committed and then refresh the website
